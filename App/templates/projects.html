AWS Cloud Resume Challenge

Took advantage of various AWS services such as Lambda, S3, DynamoDB and Cloudfront. Added a visitor counter using Python and Javascript, storing data in DynamoDB. Created a Lambda function using Python and boto3 library, acting as an API to manage interactions between frontend and DynamoDB. Used Terraform to define and deploy AWS resources. Implemented CI/CD for frontend and backend.

AWS 3-Tier Architecture

3-Tier architecture deployed on AWS using Terraform - Consists of a VPC covering 2 AZs. Each AZ has 1 Public and 2 Private Subnets. Public Subnet - NAT Gateway and EC2 instance hosting Apache Web Server. Private Subnet - EC2 Instance that accesses internet through SSH via NAT Gateway.
